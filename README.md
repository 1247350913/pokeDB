<div align="center">
  <h1>pokeDB</h1>
</div>
This is a web app to find a compare niche HM-slave pokemon! You can filter by Type, Game/Generation, HMs, and even sort the results by base stats.

<div align="center">
  <h2>Application Features</h2>
</div>

### Home Page
This page displays all pokemon by default until filters or parties are selected.<br>

<div align="center">
  <h2>Developer Resources</h2>
</div>

### Requirements
- JavaScript -- 3.10
- Node.js -- 6.13

For detailed requirements and dependencies, see package.json

### Development

#### Installing Dependencies
From within the root directory:
```sh
npm install
```

#### Running the app in a development environment
From within the root directory:
```sh
npm run build
npm run client-dev
npm run server-dev
```
